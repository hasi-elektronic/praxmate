# Praxmate

**Das Terminbuchungssystem für Arztpraxen.**

Ein Produkt von Hasi Elektronic · Vaihingen an der Enz

## Über

Praxmate ist ein schlankes, DSGVO-konformes Terminbuchungs-Widget, das direkt auf der Website Ihrer Arztpraxis läuft. Keine monatlichen Fesseln an Doctolib. Keine Daten im Ausland. Nur Ihre Praxis, Ihre Marke, Ihre Kontrolle.

- 🏥 Embed direkt in Ihre bestehende Website
- 🛡️ Server in Frankfurt, AV-Vertrag inklusive
- 📅 E-Mail & SMS-Erinnerungen
- 💰 Ab 39 €/Monat (statt 129 € bei Doctolib)
